Thank you for installing Datasance PoT.

This chart deploys:
- The Datasance ioFog operator
- A ControlPlane custom resource instance

To verify:
  kubectl get pods -n {{ .Release.Namespace }} -l {{ include "pot.selectorLabels" . | replace "\n" "," }}
  kubectl get controlplanes.datasance.com -n {{ .Release.Namespace }}

If you need to disable the ControlPlane instance creation, set controlplane.create=false.
